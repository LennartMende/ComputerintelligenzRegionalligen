# Kompassmodell Regionalliga

Dieses Projekt wurde im Rahmen des Masterstudiums im Modul  
**„Naturinspirierte Methoden der Computerintelligenz“** entwickelt.

Ziel ist die Anwendung naturinspirierter Optimierungs- und Entscheidungsverfahren auf ein geografisch-strukturiertes Problem aus dem Fußballkontext. Konkret werden Vereine der Regionalliga anhand ihrer Standortdaten modelliert und im Sinne eines „Kompassmodells“ analysiert. Die Fußballvereine sollen in vier Regionalligen mit je 20 Teams eingeteilt werden. Dabei soll der Fahrtweg innerhalb einer Liga entsprechend der festgelegten Metriken minimiert werden.

Der Fokus liegt dabei auf der Verbindung von heuristischen Methoden, räumlicher Strukturierung und datengetriebenem Problemlösen.

Neben der numerischen Analyse der gefundenen Lösungen wird auch eine graphische Darstellung der Staffeleinteilungen implementiert.

Der oben dargestellte Anwendungsfall ist der zentrale Aspekt der Implementierung. Jedoch wurde so flexibel wie möglich programmiert, damit ähnliche Probleme lösbar sind, wie beispielsweise die Einteilung in eine andere Staffelanzahl oder die Gruppierung zufällig generierter Punkte.
